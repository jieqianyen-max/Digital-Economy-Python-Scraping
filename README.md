# 📊 数字经济数据采集与分析实战项目

> **项目目标**：对标金融科技与数字经济标准，实现从静态网页到动态 API 接口的全自动化数据采集与资产化。

## 🚀 已实现的技术突破 (Technical Milestones)

### 1. 破解反爬拦截
* **Debugger 破解**：通过浏览器开发者工具（DevTools）禁用硬编码断点，成功绕过雪球网等主流金融网站的 `debugger` 防调试陷阱。
* **身份伪装**：封装 `User-Agent` 与 `Headers` 请求头，模拟真实用户浏览器行为。

### 2. 深层数据接口嗅探 (API Sniffing)
* 弃用低效的 HTML 网页解析（BeautifulSoup）。
* 深入 **Network (网络)** 面板，直接抓取后端 `JSON` 异步数据接口，实现极速、纯净的数据获取。

### 3. 数据资产自动化转化
* 使用 **Pandas** 库对抓取的非结构化数据进行清洗。
* 实现数据自动导出为 `.xlsx` (Excel) 格式，形成可直接用于商业分析的“数字资产”。

### 4. 会话身份伪装：
* 掌握了 Cookie 注入 技术，实现了带权接口的模拟登录访问，突破了游客访问频率限制。

---

## 📂 文件仓库说明
* `diyi.py`: 基础静态网页抓取实战（Quotes 练习网站）。
* `day2.py`: 雪球网金融数据接口逆向抓取脚本。
* `*.xlsx`: 自动化采集生成的结构化数据成果。
